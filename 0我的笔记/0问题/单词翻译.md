### 🔹 **square**

- **中文**：平方
- **含义**：表示一个数的平方（如 `x²`），或指“正方形矩阵”。
- **用法示例**：
    - `square = x * x`
    - 在矩阵问题中，`n x n` 的矩阵也叫 square matrix。

---

### 🔹 **matrix**

- **中文**：矩阵
- **含义**：二维数组，常用于表示表格、图像、图等。
- **用法示例**：
```
    matrix = [[1,2],[3,4]]
```
---

### 🔹 **fib**

- **中文**：斐波那契（Fibonacci）
- **含义**：斐波那契数列，递推公式：`fib(n) = fib(n-1) + fib(n-2)`
- **用法示例**：
    
```
    def fib(n):
        if n <= 1: return n
        return fib(n-1) + fib(n-2)
```
---

### 🔹 **flatten**

- **中文**：扁平化
- **含义**：将嵌套结构（如二维数组）变成一维列表。
- **用法示例**：
```
    [[1,2], [3,4]] → [1,2,3,4]
```
    
    常见于处理多维数据。

### 🔹 **sieve_of_eratosthenes**

- **中文**：埃拉托斯特尼筛法
- **含义**：一种高效找出所有小于等于 N 的素数的算法。
- **用法示例**：
```
    def sieve_of_eratosthenes(n):
        primes = [True]*(n+1)
        for i in range(2, int(n**0.5)+1):
            if primes[i]:
                for j in range(i*i, n+1, i):
                    primes[j] = False
        return [i for i in range(2,n+1) if primes[i]]
```

### 🔹 **prime**

- **中文**：素数
- **含义**：只能被 1 和自身整除的大于 1 的自然数。
- **用法示例**：
```
    is_prime(7) -> True
```

### 🔹 **stack**
- **中文**：栈
- **含义**：后进先出（LIFO）的数据结构。
- **用法示例**：
```
    stack = []
    stack.append(1)   # 入栈
    stack.pop()       # 出栈，返回 1
```
---

### 🔹 **deque**

- **中文**：双端队列（double-ended queue）
- **含义**：可以在两端进行插入和删除操作的队列。
- **用法示例**：
```
    from collections import deque
    dq = deque()
    dq.appendleft(1)  # 左端入
    dq.append(2)      # 右端入
```

---

### 🔹 **eval_rpn**

- **中文**：评估逆波兰表达式
- **含义**：计算 RPN（Reverse Polish Notation）表达式的值。
- **用法示例**：
```
    eval_rpn(["2", "1", "+", "3", "*"])  # 返回 9
```

---

### 🔹 **tokens**

- **中文**：标记 / 词元
- **含义**：将字符串拆分成一个个有意义的部分，比如数字和运算符。
- **用法示例**：
```
    tokens = ["2", "1", "+", "3", "*"]
```

---

### 🔹 **ops**

- **中文**：操作符
- **含义**：存放运算符对应函数的字典。
- **用法示例**：
```
    ops = {
        '+': lambda a,b: a+b,
        '-': lambda a,b: a-b,
        '*': lambda a,b: a*b,
        '/': lambda a,b: int(a/b)
    }
```
---

### 🔹 **generate_spiral_matrix**

- **中文**：生成螺旋矩阵
- **含义**：按顺时针方向从外到内填充数字，形成螺旋形矩阵。
- **用法示例**
```
    generate_spiral_matrix(3) →
    [
      [1, 2, 3],
      [8, 9, 4],
      [7, 6, 5]
    ]
```
---
