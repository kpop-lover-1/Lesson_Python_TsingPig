### ✅ 1. 官方 & 权威文档

#### 🔹 Linux 命令行基础

- **The Linux Command Line (TLCL) 中文版**（免费在线书）

    [https://www.gnu.org/software/bash/manual/](https://www.gnu.org/software/bash/manual/)  
    👉 深度参考，适合查语法（如变量、循环、函数）
    

#### 🔹 命令速查

- **tldr-pages（你已安装！）**
    ```
    tldr ls      # 查看 ls 用法
    tldr grep    # 查看文本搜索
    tldr find    # 查找文件
    tldr chmod   # 权限管理
    ```
    
    👉 比 `man` 更简洁，适合初学者。

### ✅ 3. 实践建议

1. **在 Windows 上安装 WSL2（推荐 Ubuntu）**  
    → 真实 Linux 环境，完美支持所有命令  
    安装指南：[https://learn.microsoft.com/zh-cn/windows/wsl/install](https://learn.microsoft.com/zh-cn/windows/wsl/install)
    
2. **每天练 5 个命令**  
    例如：`ls`, `cd`, `pwd`, `cp`, `mv`, `rm`, `cat`, `grep`, `ps`, `kill`, `chmod`, `tar`
    
3. **写简单 Shell 脚本**  
    例如：备份文件、批量重命名、统计日志行数

## 三、整合建议：如何安排时间？

|时间|内容|
|---|---|
|**第1-2周**|Python 基础语法 + 蓝桥杯简单题（模拟、字符串）|

> - 模拟（日期、时钟、进制转换）

> - 数学（最大公约数、素数、排列组合）
> - 简单 DP / DFS（近年有增加趋势）
