## ✅ 一、Python 中 `%` 的定义
在 Python 中，**取模运算 `a % b` 的结果始终与除数 `b` 同号（或为 0）**，并且满足以下恒等式：
> **`a == (a // b) * b + (a % b)`**

其中：
- `//` 是**向下取整除法**（floor division）
- `%` 是取模（modulo）
这个规则决定了负数取模的行为。

---

## ✅ 二、举个例子：`-1 % 4`
我们手动计算 `-1 % 4`：
### 第一步：计算 `-1 // 4`
- 普通除法：`-1 / 4 = -0.25`
- **向下取整**（floor）：往更小的整数方向取 → `-1`
    - 注意：不是“四舍五入”，也不是“向零取整”！
    - 所以 `-1 // 4 = -1`

### 第二步：代入公式
```
a % b = a - (a // b) * b
```
所以：
```
-1 % 4 = -1 - (-1) * 4 = -1 + 4 = 3
```

✅ 结果是 **3**

👉 规律：负数对正数取模，结果总是在 `[0, b-1]` 范围内！

## ✅ 四、为什么这样设计？

Python 这样设计是为了让 `%` 在**循环、索引、周期性问题**中更自然。

比如你有一个长度为 4 的列表，想从末尾往前走：

```
arr = ['A', 'B', 'C', 'D']
# 倒数第1个：arr[-1] → 'D'
# 用正索引表示：(-1) % 4 = 3 → arr[3] → 'D'
```

所以：

- `-1 % n` 自动变成 `n-1`
- `-2 % n` 变成 `n-2`
- ……

这正是**环形数组**需要的行为！

## ✅ 五、对比其他语言（重要！）

不同语言对负数取模的处理**不一样**：

|语言|`-1 % 4`的结果|说明|
|---|---|---|
|**Python**|**3**|结果 ≥ 0，符合数学模运算|
|C / C++ / Java|**-1**|结果符号与被除数相同|
|JavaScript|**-1**|同上|

所以在 Python 中，`%` 更适合做**循环索引**！

## ✅ 七、应用：环形数组索引

```
n = 5
for i in range(n):
    left = (i - 1) % n   # 左邻居
    right = (i + 1) % n  # 右邻居
    print(f"i={i}, left={left}, right={right}")
```

输出：

```
i=0, left=4, right=1
i=1, left=0, right=2
i=2, left=1, right=3
i=3, left=2, right=4
i=4, left=3, right=0
```

✅ 完美实现环形连接！

## ✅ 一、先看公式：`a % b = a - (a // b) * b`

这是 Python 中取模运算的**数学定义**。

### 🔍 每个部分的意思：

- `a % b`：是你要算的结果（余数）
- `a // b`：是整除，即向下取整除法
- `(a // b) * b`：是能被 `b` 整除的最大倍数
- `a - ...`：就是剩下的部分，也就是余数

---

## ✅ 二、那 `a // b` 是怎么计算的？

### 🎯 关键点：`//` 是 **向下取整除法**（floor division）

不是“四舍五入”，也不是“向零取整”！

#### ✅ 规则：

> 对于任意实数 `x`，`floor(x)` 是小于等于 `x` 的最大整数。

### 💡 举几个例子：

|表达式|计算过程|结果|
|---|---|---|
|`5 // 2`|`5 / 2 = 2.5` → 向下取整 → `2`|`2`|
|`4 // 3`|`4 / 3 ≈ 1.33` → 向下取整 → `1`|`1`|
|`-1 // 4`|`-1 / 4 = -0.25` → 向下取整 → `-1`|`-1`|
|`-5 // 3`|`-5 / 3 ≈ -1.67` → 向下取整 → `-2`|`-2`|

👉 注意：负数时，结果比实际值更小！

---

## ✅ 三、现在我们用公式验证：`-1 % 4`

### 第一步：计算 `a // b`

- `a = -1`, `b = 4`
- `-1 / 4 = -0.25`
- 向下取整 → `-1`
- 所以：`-1 // 4 = -1`

### 第二步：代入公式

```
-1 % 4 = -1 - (-1) * 4
       = -1 + 4
       = 3
```

✅ 结果是 **3**

## ✅ 六、总结：如何计算负数取模？

### 步骤如下：

1. **先算 `a // b`**：向下取整除法
2. **再用公式**：`a % b = a - (a // b) * b`
3. **结果一定在 `[0, b-1]` 范围内**（当 `b > 0`）
