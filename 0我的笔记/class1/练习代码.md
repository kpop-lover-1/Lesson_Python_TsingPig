### 🔹 1. 变量与数据类型

**题目**：创建一个整数 `a = 100`，一个浮点数 `b = 3.14`，一个字符串 `s = "Hello"`，一个列表 `lst = [1, 2, 3]`。尝试修改列表中的元素，并验证 `int`、`str` 是否真的不可变。

```
a = 100
b = 3.14
s = "Hello"
lst = [1, 2, 3]

# 修改列表（可变）
lst[0] = 999
print("修改后列表:", lst)  # [999, 2, 3]

# 尝试“修改”字符串（实际是创建新对象）
s_new = s + " World"
print("原字符串 s:", s)      # Hello（未变）
print("新字符串 s_new:", s_new)  # Hello World

# 验证 id 是否变化
print("s 的 id:", id(s))
print("s_new 的 id:", id(s_new))  # 不同 → 不可变
```

### 🔹 2. 字符串切片与操作

**题目**：给定字符串 `text = "abcdefghijk"`，完成以下操作：

- 输出从第3个字符到倒数第2个字符
- 每隔2个字符取一个（即步长为3）
- 将字符串反转
- 判断是否以 `"abc"` 开头

```
text = "abcdefghijk"

print(text[2:-1])       # cdefghij
print(text[::3])        # adgj
print(text[::-1])       # kjihgfedcba
print(text.startswith("abc"))  # True
```

### 🔹 4. 多行输入构建矩阵

**题目**：先输入两个整数 `n` 和 `m`，表示矩阵行数和列数。然后读入 `n` 行，每行 `m` 个整数，构建二维列表并打印。

```
n, m = map(int, input().split())
matrix = []
for _ in range(n):
    row = list(map(int, input().split()))
    matrix.append(row)

# 或用列表推导式：
# matrix = [list(map(int, input().split())) for _ in range(n)]

for row in matrix:
    print(row)
```

### 🔹 5. 排序与 Lambda

**题目**：给定列表 `words = ["banana", "apple", "cherry", "date"]`，按以下规则排序并输出：

- 按单词长度升序
- 若长度相同，按字母顺序升序（提示：Python 默认元组比较）

```
words = ["banana", "apple", "cherry", "date"]
sorted_words = sorted(words, key=lambda x: (len(x), x))
print(sorted_words)
# 输出: ['date', 'apple', 'banana', 'cherry']
```

### 🔹 6. 函数定义：默认参数 + 可变参数

**题目**：定义一个函数 `multiply(base, *args, factor=1)`，功能是将 `base` 与所有 `args` 相乘后再乘以 `factor`。测试调用：

- `multiply(2, 3, 4)`
- `multiply(2, 3, 4, factor=10)`

```
def multiply(base, *args, factor=1):
    result = base
    for x in args:
        result *= x
    return result * factor

print(multiply(2, 3, 4))           # 2*3*4*1 = 24
print(multiply(2, 3, 4, factor=10))  # 2*3*4*10 = 240
```

### 🔹 7. 使用快读模板处理大量输入

**题目**：模拟竞赛场景——先输入一个整数 `n`，然后输入 `n` 行字符串，每行是一个单词。统计所有单词中元音字母（a, e, i, o, u）出现的总次数。

> ✅ 要求：使用快读模板提升效率

```
import sys
input = lambda: sys.stdin.readline().strip()

n = int(input())
vowels = set('aeiou')
total = 0

for _ in range(n):
    word = input().lower()
    for char in word:
        if char in vowels:
            total += 1

print(total)
```

### 🔹 8. 综合挑战：自定义排序字典

**题目**：有一个字典 `scores = {"Alice": 85, "Bob": 92, "Charlie": 78, "Diana": 92}`。要求按分数**降序**排序，若分数相同则按名字**升序**排序，输出排序后的名字列表。

```
scores = {"Alice": 85, "Bob": 92, "Charlie": 78, "Diana": 92}

# 方法1：sorted + lambda
result = sorted(scores.items(), key=lambda x: (-x[1], x[0]))
names = [name for name, score in result]
print(names)  # ['Bob', 'Diana', 'Alice', 'Charlie']

# 方法2：直接对 keys 排序
names2 = sorted(scores.keys(), key=lambda name: (-scores[name], name))
print(names2)
```
